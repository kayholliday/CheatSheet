Semantic HTML

Table of Tags
Name Tag Description
Header <header> A container usually for navigational links or intro content containing <h1> to <h6>.
Nav <nav> Defines a block of navigation linke e.g. menus and tables of content. It can be used in the header element or on its own.
Main <main> This encapsulates the dominant content within the webpage and makes it easier to read.
Footer <footer> This is the content at the bottom of the subject information. It contains things like contact info, copyright info etc..
Article <article> This element holds content that makes sense on its own.
Section <section> Defines elements such as chapters, headings or any other area of the document.
Aside <aside> Information inside this element enhances the info in an article element but it is not required to undersand it.
Figure <figure> An element used to encapsulate a media, such as images or illustrations.
Figcaption <figcaption> This describes the media inside the figure element. This is different to <p> tag because the text will follow the figure if it is moved.
Audio <audio> This element is used to embed audio content into a document. It uses src to link to the source.
Video <video> This element is used to embed video content.
Embed <embed> This element can be used for files or external links and different media. It is self closing and deprecated.

Table of Attributes
Name Attribute Description
Source src This provides the link to the source of the media being embedded.
Autoplay autoplay This results in video automatically playing when loaded.
Controls controls This adds play/pause onto the video.
Loop loop This results in the video continuously playing.
